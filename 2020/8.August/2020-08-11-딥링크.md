# 딥링크

#### ✔️ 딥링크란?

`특정 페이지에 도달 할 수 있는 링크`

#### ✔️ 모바일 딥링크란?

`모바일 어플리케이션의 특정 페이지에 도달 할 수 있는 링크`

즉, 모바일 어플리케이션에도 특정 페이지에 들어갈 수 있는 링크가 있는 것.

- 웹 어플리케이션이 http://나 https://프로토콜을 갖고 있는 것처럼, 모바일 어플리케이션도 각자의 프로토콜을 갖고 있음.

  ```
  유튜브: youtube://
  멜론: meloniphone://
  ```

  해당 링크를 누르면 네이티브 앱으로 바로 이동하는 것이 `모바일 딥링크`
