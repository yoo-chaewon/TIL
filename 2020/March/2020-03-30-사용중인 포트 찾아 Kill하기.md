## [MAC] 사용중인 포트 찾아서, Kill하기

- 찾기

  ```bash
  lsof -i :포트번호
  ```

- 결과

  ```bash
  > lsof -i :8000
  
  
  COMMAND   PID       USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
  java    219-- yoochaewon   80u  IPv6 0xab22----      0t0  TCP *:irdmi (LISTEN)
  ```

  



- Kill 하기

  ```bash
  kill -9 PID번호
  ```

- Ex)

  ```bash
  kill -9 1233
  ```
